# 📘 C++ 算法竞赛速查笔记：数组与 Vector 核心操作

在算法竞赛中，熟练运用 STL（Standard Template Library）能极大缩短代码编写时间，并减少手写 `for` 循环带来的边界 Bug。本笔记默认已引入 `<vector>` 和 `<algorithm>` 头文件，且使用 `using namespace std;`。

## 一、 增 (Insert & Append)

竞赛中，避免使用低效的单点插入，多利用尾部追加和区间插入。

### 1. 尾部追加 (最常用)

- **语法:** `v.push_back(x)`
    
- **时间复杂度:** 均摊 $O(1)$
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v;
    for(int i = 0; i < 5; i++) {
        v.push_back(i); // 将 0~4 依次塞入尾部
    }
    ```
    

### 2. 区间批量插入 (代替低效的 for 循环)

- **语法:** `v.insert(插入点迭代器, 起始迭代器, 终止迭代器)`
    
- **时间复杂度:** $O(N + M)$ （底层的元素集体平移只需一次）
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 2, 8, 9};
    int arr[] = {3, 4, 5};
    // 在 8 的前面（即 v.begin() + 2）一次性插入 arr 的所有元素
    v.insert(v.begin() + 2, arr, arr + 3); 
    // 结果: {1, 2, 3, 4, 5, 8, 9}
    ```

## 二、 删 (Erase & Remove)

删除操作涉及内存移动，需谨慎处理迭代器失效问题。

### 1. 尾部删除

- **语法:** `v.pop_back()`
    
- **时间复杂度:** $O(1)$。常用于模拟栈结构。
    

### 2. 区间删除 (慎用单点删除)

- **语法:** `v.erase(起始迭代器, 终止迭代器)`
    
- **时间复杂度:** $O(N)$
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 2, 3, 4, 5};
    // 删除 2 和 3 (左闭右开区间)
    v.erase(v.begin() + 1, v.begin() + 3); 
    // 结果: {1, 4, 5}
    ```

## 三、 改与初始化 (Update & Initialize)

多组测试数据（Multiple Test Cases）下，快速重置状态是刚需。

### 1. 批量重置 (Fill)

- **功能:** 将某段区间全部替换为同一个特定值。
    
- **语法:** `fill(起始, 终止, val)`
    
- **代码示例:**
    
    C++
    
    ```
    int arr[100];
    fill(arr, arr + 100, -1); // 极其常用：把原生数组全部初始化为 -1
    
    vector<int> v(50, 1);
    fill(v.begin(), v.begin() + 10, 0); // 只把前 10 个元素变成 0
    ```
    

### 2. 推翻重构 (Assign)

- **功能:** 销毁 `vector` 原有数据，直接用新长度和新值覆盖。
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 2, 3};
    v.assign(10, 0); // 原来的 1,2,3 消失，变成 10 个 0
    ```

## 四、 查与统计 (Search & Count)

绝对不要自己写 `for` 循环去找最大值或统计个数，用库函数不仅代码短，常数也更小。

### 1. 统计特定值出现次数

- **语法:** `count(起始, 终止, 目标值)`
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 2, 2, 3, 2};
    int num = count(v.begin(), v.end(), 2); // 返回 3
    ```
    

### 2. 查找极值 (Max/Min Element)

- **注意:** 返回的是**迭代器/指针**，取具体值需加 `*`。
    
- **代码示例:**
    
    C++
    
    ```
    int arr[] = {5, 2, 9, 1};
    int max_val = *max_element(arr, arr + 4); // 返回 9
    // 技巧：求极值的下标
    int max_idx = max_element(arr, arr + 4) - arr; // 返回 2
    ```
    

### 3. 二分查找 (神级算法，必须掌握)

- **前提:** 数组必须**已经升序排序**。时间复杂度 $O(\log N)$。
    
- `lower_bound`: 找第一个 **$\ge$** 目标值的位置。
    
- `upper_bound`: 找第一个 **$>$** 目标值的位置。
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 3, 3, 5, 7};
    // 找第一个 >= 3 的数字下标
    int idx1 = lower_bound(v.begin(), v.end(), 3) - v.begin(); // 返回 1
    
    // 找第一个 > 3 的数字下标
    int idx2 = upper_bound(v.begin(), v.end(), 3) - v.begin(); // 返回 3
    ```

## 五、 竞赛高阶神技 (Advanced Tricks)

这部分是 C++ 选手与其他语言选手拉开代码长度差距的核心。

### 1. 数组去重 (Unique)

- **原理:** `unique` 只能去除**相邻**的重复元素，因此使用前**必须先排序**。它会把不重复的元素移到数组前部，并返回去重后新序列末尾的迭代器。
    
- **标准连招:** 排序 -> unique -> erase
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {4, 1, 2, 2, 1, 3, 4};
    
    // 1. 先排序：{1, 1, 2, 2, 3, 4, 4}
    sort(v.begin(), v.end()); 
    
    // 2. unique 移动不重复元素，erase 删掉后面的废弃尾巴
    v.erase(unique(v.begin(), v.end()), v.end()); 
    
    // 结果: {1, 2, 3, 4}
    ```
    

### 2. 字典序全排列 (Next Permutation)

- **功能:** 自动生成当前数组的下一个字典序排列。如果当前是最大排列（如 `4, 3, 2, 1`），则返回 `false` 并重置为最小排列。暴力杯（蓝桥杯）防挂分利器，替代手写 DFS。
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 2, 3}; // 必须从最小字典序开始
    
    do {
        // 输出当前排列：123, 132, 213, 231, 312, 321
        for(int x : v) cout << x; 
        cout << "\n";
    } while (next_permutation(v.begin(), v.end()));
    ```
    

### 3. 区间反转 (Reverse)

- **功能:** 物理翻转某一段区间。
    
- **代码示例:**
    
    C++
    
    ```
    vector<int> v = {1, 2, 3, 4, 5};
    reverse(v.begin(), v.end()); // 结果: {5, 4, 3, 2, 1}
    ```

## ⚠️ 竞赛避坑核心提示

1. **防止 TLE（超时）：** 知道元素大概数量时，先写 `v.reserve(N)`，避免 `vector` 内部频繁扩容搬家。
    
2. **函数传参致命伤：** 当你把 `vector` 作为函数参数传递时，**一定、必须**加上 `&` 符号（引用传递），如 `void dfs(vector<int>& v)`。否则每次递归都会复制一遍整个数组，瞬间超时爆内存。
    
3. **size() 的陷阱：** `v.size()` 返回的是无符号整数（`size_t`）。如果 `v` 为空，`v.size() - 1` 会变成一个巨大的正数导致越界，请养成写 `(int)v.size() - 1` 或判空的习惯。
