### 1. 单引号 (`' '`)：字符字面量 (Character Literal)

- **数据类型**：`char`（字符型）。
    
- **存放内容**：只能存放**一个**字符（包括字母、数字、符号或转义字符）。
    
- **内存占用**：通常占用 **1 个字节**（8 位）。它在底层其实就是一个整数（对应 ASCII 码）。
    
- **示例**：
    
    C++
    
    ```
    char a = 'A';  // 字母 A (ASCII码为 65)
    char b = '1';  // 数字字符 1 (ASCII码为 49)
    char c = '\n'; // 转义字符：换行 (占用 1 个字符空间)
    ```
    
- **错误写法**：`char x = 'AB';` （不能放多个字符，编译器会报错或截断）。
    

### 2. 双引号 (`" "`)：字符串字面量 (String Literal)

- **数据类型**：常量字符数组 `const char[]`（或者 C++ 中的 `std::string`）。
    
- **存放内容**：可以存放零个、一个或多个字符的组合。
    
- **内存占用**：字符的实际数量 **+ 1 个字节**。
    
    - **为什么加 1？** 因为 C/C++ 规定，双引号引起来的字符串，编译器会自动在末尾加上一个隐藏的字符串结束标志：`'\0'`（空字符，ASCII码为 0）。
        
- **示例**：
    
    C++
    
    ```
    const char* str1 = "Hello"; // 包含 H, e, l, l, o, \0 (共 6 个字节)
    std::string str2 = "C++";   // 包含 C, +, +, \0 (共 4 个字节)
    ```
    

---

### 3. 最经典的对比：`'A'` vs `"A"`

很多初学者会混淆这两个概念，认为它们是一样的，但它们在内存里完全不同：

|**特性**|**'A' (单引号)**|**"A" (双引号)**|
|---|---|---|
|**类型**|`char`|`const char[2]` (字符数组)|
|**内存大小**|1 字节|2 字节|
|**内存内容**|`[ 'A' ]`|`[ 'A', '\0' ]`|
|**本质**|整数 65|一个内存地址（指向该数组的首地址）|

### 4. 常见踩坑与注意事项

**坑 1：类型不匹配导致报错**

如果你尝试把双引号赋值给一个 `char` 变量，或者把单引号当作字符串传递，就会报错：

C++

```
char x = "A";  // ❌ 错误：不能把 const char* (指针) 赋值给 char
string y = 'A'; // ❌ 错误：通常不能直接用单个 char 初始化 string（需要用其他构造方式，如 string(1, 'A')）
```

**坑 2：比较大小的逻辑不同**

- 单引号比较的是**数值**：`'A' == 65` 是成立的（`true`）。
    
- 双引号比较的是**地址**或**字符串内容**：`"A" == "B"` 比较的是两个指针地址（在 C 风格中），或者通过 `std::string` 重载的运算符来比较内容。
    

**拓展：C++11 之后的新玩法（原生字符串）**

如果你有一大段文本，里面既包含单引号又包含双引号，转义起来会非常麻烦（比如 `\"` 和 `\'`）。C++11 引入了 `R"( )"` 语法，忽略所有转义字符：

C++

```
std::string text = R"(He said: "It's C++!")"; 
// 括号里的内容原样输出，不需要写成 "He said: \"It\'s C++!\""
```
