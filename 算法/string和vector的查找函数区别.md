
| **需求**    | **对于 string 怎么写？** | **对于 vector 怎么写？**                                        | **返回值**                                                |
| --------- | ------------------ | --------------------------------------------------------- | ------------------------------------------------------ |
| **找单个元素** | `s.find('a')`      | `std::find(v.begin(), v.end(), 5)`                        | `string`: 下标 (`size_t`)<br><br>  <br><br>`vector`: 迭代器 |
| **找一串元素** | `s.find("abc")`    | `std::search(v.begin(), v.end(), sub.begin(), sub.end())` | `string`: 下标 (`size_t`)<br><br>  <br><br>`vector`: 迭代器 |
