### 为什么能加减
一般数组的例如`arr[10]`就是`arr[arr.begin()+10]`所以能加减

### vector和普通数组差异
#### 什么能用
**arr刚开始的内存定好了就无法改变**
普通数组没有`arr[i].begin()`等.但是有些情况能用`arr`替代`arr.begin()`
但像 `reverse()`、`sort()`、`lower_bound()` 这些函数，它们并不属于 `vector`，而是独立存在于 `<algorithm>` 头文件里的**通用算法**。
这些可以
```
// 对前 n 个元素排序 
sort(arr, arr + n); // 完全等价于 sort(v.begin(), v.end()) 

// 反转前 n 个元素 
reverse(arr, arr + n); // 完全等价于 reverse(v.begin(), v.end())
```
#### 插入（insert）和删除（erase）能用吗？（结构类）

**绝对不能用！**

这里要分清**“通用算法”**和**“容器专属招式”**的区别：

- `sort` 和 `reverse` 只是在已有的格子里互相交换数据，不改变格子的总数量。
    
- `insert` 和 `erase` 是要**改变内存长度**的。
    

`int arr[10]` 在声明的那一刻起，它在内存里就永远只有 10 个格子（静态分配）。你无法无中生有地硬挤进去第 11 个格子（不支持扩容），也无法把物理内存挖掉一块（不支持缩容）。 而 `vector` 内部写了极其复杂的代码，当你 `insert` 导致空间不够时，它会在后台偷偷找一块更大的新内存，把老数据搬过去，这就是所谓的“动态数组”。原生数组没有这个超能力。
